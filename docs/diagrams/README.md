# Diagrams Directory

This directory contains visual diagrams for the YatriSetu Smart Transit Platform.

## Files

| File | Description |
|------|-------------|
| use-case-diagram.png | System use case diagram showing actor interactions |

## Adding the Use Case Diagram

To add the use case diagram image:

1. Save the use case diagram image as `use-case-diagram.png`
2. Place it in this directory (`docs/diagrams/`)
3. The diagram will be automatically referenced in `docs/USE_CASE_DIAGRAM.md`

## Image Specifications

| Property | Specification |
|----------|---------------|
| Format | PNG (recommended) or JPG |
| Resolution | Minimum 1920x1080 pixels |
| File Size | Maximum 2MB |
| Color Mode | RGB |
